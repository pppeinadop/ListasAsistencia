¿Qué es ListasAsistencia?

Es una herramienta para crear y administrar listas de asistencia para clases, reuniones o eventos. Diseñada para ser simple, rápida y accesible desde cualquier dispositivo.

Rápido

Interfaz ligera y enfocada en la velocidad.

Exportable

Exporta las listas a CSV o PDF.

Compartible

Comparte enlaces para que otros accedan a las listas.

¿Listo para empezar?

  1. Clona el repositorio.
  2. Abre index.html en tu navegador o publica con GitHub Pages.
  3. Integra con tu backend si necesitas guardar datos.
